String processor Open on Hyperskill

Write a program that emulates a string processor. It reads two strings and an operation and outputs a result.

There are three possible operations:
  • equals - it returns true if these strings are equals, otherwise false;
  • plus - it appends the second line to the first line;
  • endsWith - it returns true if the first string ends with the second string.

The program should output the result as a string. If the operation is unknown, the program should print "Unknown operation" without quotes.


Sample Input:
abc
equals
cde
Sample Output:
false

Sample Input:
abbc
plus
bb
Sample Output:
abbcbb

Sample Input:
abbc
endsWith
bc
Sample Output:
true


Memory limit: 256 Mb
Time limit: 5s
